# SyncNote

A collaborative note-taking Android application with real-time synchronization powered by Firebase.

**University Project**  
**Student ID:** 2207103  
**Name:** Md. Shahidur Rahman

---

## Features

### Authentication
- User registration and login with email
- Password recovery via security questions
- Secure session management

### Note Management
- Create, edit, and delete notes
- Rich text formatting (bold, italic, underline, strikethrough)
- Text alignment options (left, center, right)
- Bulleted and numbered lists
- Note color customization

### Organization
- Categorize notes into folders
- Search notes by title or content
- Sort notes by date or title

### Collaboration
- Share notes with other users
- Permission levels (View / Edit)
- Manage shared access
- Real-time sync across devices

### Settings
- Dark/Light theme support
- Account management
- Security settings

---

## Tech Stack

- **Language:** Java
- **Min SDK:** 23 (Android 6.0)
- **Target SDK:** 36
- **Backend:** Firebase Realtime Database
- **Authentication:** Firebase Auth
- **UI:** Material Design Components
